I like to build things. Here are some notable projects.

Lan File Sharing System

[repo]

A desktop application uses an IP scanner to scan the whole LAN and allows user to share Files/Folders from their own PCs.

Main feature of this application is that each PC user can stream media files(.mp3, .mp4, etc.) without downloading it from another PC users. An user can also dowload files from other user's PC.

2048 AI

[repo]

2048-AI is a artificial intelligence implemented version. Here AI solves the game within shortest possible times and steps using heuristic algorithm named alpha-beta pruning. A Javascript implementation of the algorithm shows 80% accurate to solve the puzzle.

Mega Structure Modeling: Burj Al Arab

[repo]

This application specifies a mega structure 3D modeling named "Bhuj Al Arab" using opengl libraries.The modeling design contains the features like modeling, walking camera, texturing, lighting, effects etc.

Online Movie Ticket Management

[repo]

A website which allows user to book tickets from online and provides new updates on upcoming movies on theater. It allows user to book seat on own choice and write reviews on movies.

Tourism BD

[repo]

A mobile application that helps tourist to know famous tourist spot in Bangladesh. It also provides location and direction to tourist spot.
